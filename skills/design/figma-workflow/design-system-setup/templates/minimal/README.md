# Minimal Design System Template

A clean, focused design system for projects that need essential design foundations without complexity.

## Philosophy

Less is more. This template provides just enough structure to ensure visual consistency without overwhelming small projects.

## What's Included

### Colors
- **Primary**: Single accent color (#3B82F6 - Blue 500)
- **Neutrals**: 6-step grayscale (white, gray-100, gray-300, gray-500, gray-700, black)
- **Semantic**: Success, warning, error colors derived from primary

### Typography
- **Font Families**:
  - Sans-serif: System font stack (SF Pro, Segoe UI, Roboto)
  - Monospace: System monospace (SF Mono, Consolas, Monaco)
- **Scale**: 5 sizes (xs, sm, base, lg, xl)
- **Weights**: Regular (400), Medium (500), Bold (700)

### Spacing
- **Scale**: 5-step (4px, 8px, 16px, 24px, 48px)
- **Based on**: 4px base unit

### Components
- Button (primary, secondary, ghost variants)
- Input (text, email, password)
- Card (simple container with padding)
- Badge (status indicator)

## Best For

- MVP projects
- Internal tools
- Prototypes
- Projects with minimal UI needs
- Teams wanting flexibility to evolve design system later

## Not Ideal For

- Marketing websites requiring brand expression
- Apps with complex component libraries
- Projects needing extensive theming options
