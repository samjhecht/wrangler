# Vibrant Design System Template

A playful, energetic design system with bold colors, expressive typography, and a fun component library perfect for creative and consumer-facing applications.

## Overview

The Vibrant template brings personality and energy to your designs with a multi-color palette, bold typography, and generous spacing. It's ideal for brands that want to stand out and create memorable user experiences.

## What's Included

### Color System
- **Multi-Color Palette**: 5 vibrant color families
  - Pink (hot pink to magenta)
  - Orange (tangerine to coral)
  - Teal (bright cyan to deep teal)
  - Purple (violet to deep purple)
  - Yellow (bright yellow to gold)
- **Neutral Scale**: 9 shades for balance
- **Semantic Colors**: Vibrant success, warning, error states

### Typography
- **Font Families**:
  - Display: Poppins (bold, attention-grabbing headings)
  - Body: Work Sans (clean, readable content)
  - Accent: Fredoka One (playful highlights)
- **Font Sizes**: 10 sizes from xs (12px) to 6xl (64px)
- **Font Weights**: 4 weights optimized for impact
- **Generous Line Heights**: Extra breathing room for readability

### Spacing System
- **Generous scale**: Larger base units for spacious layouts
- 10-step scale from 2px to 160px
- 8px baseline grid for bold, open designs

### Border Radius
- **5 options**: Sharp to fully rounded
- Extra large radius (24px) for playful shapes
- Full rounds for pills and circles

### Shadows
- **6 levels**: From subtle to dramatic
- Colored shadows option for extra pop
- Strong contrast for depth

### Components (15+)
- **Buttons**: Primary, Secondary, Accent, Ghost, Outline variants with bold styling
- **Inputs**: Chunky borders, playful focus states
- **Cards**: Colorful, elevated, with personality
- **Badges**: Bright, attention-grabbing variants
- **Alerts**: Colorful, friendly messaging
- **Tooltips**: Vibrant backgrounds
- **Modals**: Eye-catching overlays
- **Tags**: Colorful chips and tags
- **Progress Bars**: Gradient-filled indicators
- **Avatars**: Colorful placeholder backgrounds
- **Tabs**: Bold active states
- **Toggles**: Playful switch designs
- **Banners**: Full-width announcement components
- **Pills**: Rounded tag alternatives
- **Callouts**: Highlighted content blocks

## Design Principles

1. **Bold & Energetic**: Colors that pop and grab attention
2. **Playful**: Rounded corners and generous spacing create friendliness
3. **Expressive**: Typography choices that convey personality
4. **Vibrant**: Multi-color palette allows for creative freedom
5. **Fun**: Components designed to delight users

## Usage

This template is ideal for:
- Consumer mobile apps
- Creative portfolios and agencies
- E-learning platforms
- Social media applications
- Youth-oriented brands
- Lifestyle and entertainment sites
- Marketing landing pages

## Customization

All tokens are defined in `tokens.json`. The vibrant color palette can be adjusted while maintaining the energetic feel. Mix and match colors freely - this template encourages creativity!

## Figma Integration

Import `figma-template.json` into Figma using the Tokens Studio plugin to create:
- Vibrant color styles for all families
- Bold text styles with display fonts
- Dramatic effect styles
- Playful component variants

## Color Palette Philosophy

Unlike traditional design systems with a single primary/secondary color, Vibrant embraces multiple color families. Use them liberally:
- Pink for CTAs and highlights
- Orange for energy and warmth
- Teal for calm interactions
- Purple for creativity
- Yellow for optimism and attention

Mix colors in the same interface to create dynamic, memorable experiences!
